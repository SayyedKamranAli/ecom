# 🛒 E-Commerce Monorepo

This monorepo contains the codebase for a complete e-commerce platform built using **Next.js**, **TypeScript**, **Tailwind CSS**, and **TurboRepo**.

## 📁 Structure

```
ecom/
│
├── e-com backend/             → Application code
├── e-com frontend/
```

## 🧰 Getting Started

### Prerequisites

- Node.js `>=18`
- npm

### Install Dependencies

```bash
npm install
```
